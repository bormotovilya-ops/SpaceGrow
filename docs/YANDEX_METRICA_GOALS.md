# Яндекс.Метрика — цели (reachGoal)

В проекте уже настроены **SPA-hit** (для навигации без перезагрузки) и отправка **целей** через `ym(..., 'reachGoal', ...)`.

## Как завести цели в Метрике

В Яндекс.Метрике откройте: **Настройки → Цели → Добавить цель → JavaScript-событие**  
И для каждой цели укажите **Идентификатор** (Goal ID) из списка ниже.

## Список целей, которые отправляет сайт

### Контакты и внешние ссылки (автоматически)

Эти цели отправляются глобально для кликов по ссылкам:

- **`contact_telegram_click`**: любой клик по `t.me` / `telegram.me`
- **`contact_phone_click`**: клик по ссылке `tel:...`
- **`contact_email_click`**: клик по ссылке `mailto:...`
- **`outbound_link_click`**: клик по внешней ссылке (другой домен)

### Основные действия (точечные цели)

- **`telegram_click`**: клик по центральной фотографии (CTA на Telegram)
- **`open_profile`**: открытие профиля (клик по аватару)
- **`open_portfolio`**: открытие портфолио
- **`open_diagnostics`**: открытие диагностики
- **`open_alchemy`**: открытие “Цифровой Алхимии”
- **`open_chat`**: открытие чат-бота

### Диагностика

- **`diagnostics_start`**: нажали “Начать диагностику”
- **`diagnostics_complete`**: дошли до экрана “Результаты диагностики”
- **`diagnostics_send_telegram`**: нажали “Обсудить план действий” (открываем Telegram с предзаполненным сообщением)

### Чат

- **`chatbot_send`**: отправили сообщение в чат-боте (на главной)
- **`profile_chat_send`**: отправили сообщение в чат-блоке на странице профиля

### Профиль

- **`profile_consultation_click`**: нажали “Получить бесплатную консультацию”

## Где это реализовано в коде

- `src/analytics/yandexMetrica.js` — `yandexMetricaReachGoal(...)` + глобальный трекинг ссылок
- `src/main.jsx` — включение глобального трекинга
- `src/components/*` — точечные вызовы целей в обработчиках кликов
