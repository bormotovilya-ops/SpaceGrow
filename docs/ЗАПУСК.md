# Как запустить приложение

## Шаг 1: Установка Node.js (если не установлен)

1. Скачайте Node.js с официального сайта: https://nodejs.org/
2. Установите LTS версию (рекомендуется)
3. Перезапустите терминал после установки

Проверьте установку:
```bash
node --version
npm --version
```

## Шаг 2: Установка зависимостей

Откройте терминал в папке проекта и выполните:

```bash
npm install
```

## Шаг 3: Запуск React приложения

```bash
npm run dev
```

Приложение откроется автоматически в браузере на адресе:
**http://localhost:3000**

### Аналитика (Яндекс.Метрика) — опционально

Метрика включается, если задан `VITE_YM_COUNTER_ID` (ID счётчика).

- Инструкция: см. [ENV_SETUP.md](./ENV_SETUP.md)

## Шаг 4: Запуск Python backend (опционально)

Если хотите использовать API:

1. Установите Python (если не установлен): https://www.python.org/
2. Установите зависимости:
```bash
cd backend
pip install -r requirements.txt
```

3. Запустите сервер:
```bash
python app.py
```

Backend будет доступен на: **http://localhost:5000**

## Быстрый старт (только фронтенд)

Если нужно просто посмотреть результат без backend:

1. `npm install`
2. `npm run dev`
3. Откройте браузер на http://localhost:3000
