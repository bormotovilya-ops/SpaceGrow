# Переключение с Groq на GigaChat

## Почему GigaChat?
- ✅ Работает в России без VPN
- ✅ Бесплатный тариф: до 10,000 токенов/день
- ✅ Отличная поддержка русского языка
- ✅ Простая настройка

## Шаг 1: Получите API ключ GigaChat

1. Зарегистрируйтесь на https://developers.sber.ru/gigachat
2. Создайте приложение
3. Получите API ключ
4. Сохраните его

## Шаг 2: Обновите .env

Добавьте в `.env`:
```
GIGACHAT_API_KEY=ваш_ключ_от_gigachat
```

## Шаг 3: Обновите код

Я обновлю `server-local.js` и `api/chat.js` для работы с GigaChat.

## Шаг 4: Перезапустите сервер

```bash
# Остановите текущий сервер (Ctrl+C)
# Запустите снова
node server-local.js
```

## Готово!

После этого чат будет работать через GigaChat вместо Groq.
