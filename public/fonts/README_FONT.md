# Шрифт Harry Potter

Для использования оригинального шрифта Harry Potter:

1. Скачайте шрифт с https://www.fontspace.com/harry-potter-font-f3199
2. Сохраните файл как `harry-potter.ttf` в папку `public/fonts/`
3. Шрифт автоматически подключится через CSS @font-face

Альтернатива: В проекте уже используется декоративный шрифт Cinzel и Cormorant Garamond из Google Fonts, который визуально похож на стиль Гарри Поттера.
