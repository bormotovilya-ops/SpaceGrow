## 3. Маркетинговые офферы

### Основное предложение: АИЦП (Автоматизированные интеллектуальные цепочки продаж)

**Определение:** АИЦП — это комплексная система, которая автоматизирует весь путь клиента от первого контакта до повторных покупок.

### Этапы цепочки продаж (воронка)

#### 1. Аудитория
**Описание:** Технологическая разведка рынка. Запускаем систему сбора данных раньше, чем вы запишете первый урок.

**Что на входе:**
- Гипотеза продукта (описание оффера, боли аудитории, ценовой сегмент)
- Трафик с предыдущего этапа (с UTM-разметкой)

**Процесс:**
- Создание прелендингов (быстрые точки захвата)
- Соединение через API/Webhooks (Make/n8n) с аналитическими таблицами
- Анализ трафика: стоимость клика, интерес аудитории, валидация гипотезы

**Технологии:**
- Fast-Deployment: Tilda / Taplink
- No-code Automation: Make / n8n
- Analytics: Pixel / Metrics
- Data Management: Google Sheets

**Результат:**
- Dashboard с реальными данными
- Стоимость клика по каждому источнику
- Интерес аудитории (процент реального взаимодействия)
- Валидация гипотезы (есть ли рынок и готовность платить)
- Сегментация (какие группы наиболее заинтересованы)
- Экономика входа (реальная стоимость привлечения)

#### 2. Лендинг
**Описание:** Одностраничный сайт, задача которого максимально эффективно превращать анонимного посетителя в потенциального клиента.

**Что на входе:**
- Валидированная гипотеза (подтверждённый оффер, ясное УТП, данные о сегментах)
- Техническое задание (логика распределения по источникам трафика)

**Технологии:**
- Tilda, Taplink, Wordpress (быстрое развёртывание)
- GetCourse (интеграция с платформой)
- Custom React (максимальная гибкость)

**Результат:**
- Технически готовый "Трамплин"
- Мгновенная передача данных (TG ID / Email / Phone) в систему автоматизации
- Бесшовная интеграция с воронкой лид-магнита
- Метрика успеха: максимальный % конверсии из клика в подписку
- Интеллектуальный фильтр: определяет, какой лид-магнит показать каждому пользователю

#### 3. Лидмагнит
**Описание:** Автоматизированная выдача ценности и первый шаг к доверию. Фильтр, а не «плюшка».

**Что на входе:**
- Живой, заинтересованный трафик (люди, которые прошли через лендинг)
- Идентификатор каждого пользователя (откуда пришли, что зацепило)

**Виды лид-магнитов:**
- PDF-гайды и чек-листы (Telegram-бот, MiniApp, сайт с автовыдачей)
- Квизы и тесты (Квиз-бот, MiniApp, кастомный квиз на сайте)
- Калькуляторы (MiniApp, сайт-калькулятор, интерактивный бот)
- Видео-уроки и вебинары (Бот с автовыдачей, приватный канал, доступ через MiniApp)
- Интерактивные инструменты (MiniApp, кастомный сайт, интерактивный бот)
- Демо-доступы и челленджи (MiniApp, личный кабинет, бот с автоворонкой)

**Результат:**
- «Утепленный» контакт (человек, который прикоснулся к продукту)
- Данные о вовлеченности

#### 4. Трипваер
**Описание:** Первая денежная транзакция. Автоматизация импульсивных покупок. Превращаем читателя в покупателя в 2 клика.

**Что на входе:**
- Прогретый лид (получил лид-магнит, прошёл автоворонку прогрева)
- Недорогой оффер (импульсивная покупка, обычно 500-2000₽)

**The Engineering:**
- Настройка бесшовного процесса: "Оплата — Фискализация — Выдача доступа"
- Всё происходит автоматически, без участия человека

**Технологии:**
- Платежные системы: ЮKassa / Prodamus / Robokassa / Юмани / TBank
- Платформа выдачи: Telegram-бот, Mini App, GetCourse

**Результат:**
- Финансовая валидация модели (реальная стоимость привлечения, конверсия, ROI)
- Профиль платёжеспособного клиента (поведенческие паттерны, готовность к покупке)
- Автоматизированный процесс продажи (готов к масштабированию)

#### 5. Автоворонки прогрева
**Описание:** Влюбление через касания. Цепочка касаний, которая подстраивается под поведение клиента и ведет его к покупке без участия.

**Что на входе:**
- История действий клиента (открытия, досматривания, паузы, клики)
- Сегментация (уровень вовлеченности, интересы, готовность к покупке)

**The Engineering:**
- Адаптивный сценарий (живой алгоритм, реагирует на каждое действие в реальном времени)
- Реагирует на открытия, учитывает досматривания, отслеживает паузы

**Технологии:**
- Конструкторы ботов: LeadTeh, Bothelp
- Кастомная разработка: Сайты, MiniApp на React, Python
- Email-сервисы: Email для GetCourse

**Результат:**
- Прогретый и лояльный клиент
- Прошел через персонализированную цепочку касаний
- Получил релевантный контент
- Доверяет и готов к покупке основного продукта
- Показывает рост конверсии из лида в клиента

**Описание:** Экосистема вашего успеха. Превращаем знания в технологичный актив. Среда, в которой доходимость до результата заложена в архитектуру.

- Архитектура доходимости: система построена так, чтобы максимальное количество клиентов досмотрели курс до конца

**Технологии:**

#### 7. Деньги
**Описание:** Масштабирование капитала. Финальный узел системы, превращающий разовые продажи в бесконечный цикл прибыли.

## 8. Ключевые термины и концепции

### АИЦП
**Автоматизированные интеллектуальные цепочки продаж** — комплексная система, которая автоматизирует весь путь клиента от первого контакта до повторных покупок.

### Этапы воронки
1. **Аудитория** — технологическая разведка рынка
2. **Лендинг** — одностраничный сайт для превращения посетителя в клиента
3. **Лидмагнит** — бесплатное предложение для сбора контактов
4. **Трипваер** — первая денежная транзакция (импульсивная покупка)
5. **Автоворонки прогрева** — автоматизированная система прогрева лидов
6. **Продукт** — основной продукт (обучающий курс или услуга)
7. **Деньги** — доход и масштабирование капитала

### Ключевые метрики
- **ROI (Return on Investment)** — возврат инвестиций
- **ROMI+ (Return on Marketing Investment)** — возврат маркетинговых инвестиций с прибылью
- **LTV (Lifetime Value)** — пожизненная ценность клиента
- **COR (Completion Rate)** — процент досматриваемости курса
- **CAC (Customer Acquisition Cost)** — стоимость привлечения клиента
